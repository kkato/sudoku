# 数独パズル

このプログラムはPythonで作成された数独パズルのプログラムです。
9x9の数独パズルの問題とその回答を生成します。

## 使用方法

1. このリポジトリをクローンします

```bash
git clone https://github.com/kkato/sudoku.git
cd sudoku
```

2. プログラムを実行します

```bash
python3 sudoku.py
```
